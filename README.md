# سرزمین راست – نسخه فارسی (mdbook)

این مخزن شامل نسخه فارسی کتاب «سرزمین راست» است که با ابزار **mdbook** ساخته شده. محتوای اصلی بر پایه پروژه [land-of-rust](https://github.com/huvaxstra/land-of-rust) نوشته شده و هدف آن ارائه یک منبع روان، ساختاریافته و به‌روز برای یادگیری زبان راست (Rust) به فارسی‌زبانان است.

## ویژگی‌ها
- ✅ ترجمه کامل و بازنویسی شده مطابق با آخرین نسخه‌های Rust
- ✅ خروجی تعاملی و قابل جستجو با پشتیبانی از قالب mdbook
- ✅ مثال‌های عملی و تمرین‌های گام‌به‌گام
- ✅ سازگار با مرورگرهای مدرن و نمایش خودکار روی GitHub Pages
